Get your State Pension

You’ll receive a letter through the post before you reach State Pension age. The letter will contain the invite code you’ll need to use this service.

If you haven’t received the letter, you’ll need to request an invite code.

You don't have to do anything if you want to delay getting your State Pension.

To get your State Pension you will need your:

  • invite code (on your Invitation to Get your State Pension letter)
  • National Insurance number
  • bank account or building society account details
