Get your State Pension

You’ll receive a letter before you reach State Pension age. It'll contain the invite code you’ll need to use this service.

If you haven’t received the letter, you’ll need to request an invite code.

You don't have to do anything if you want to delay getting your State Pension.

You will need your:

  • invite code (on your Get your State Pension letter)
  • National Insurance number
  • bank account or building society account details

You'll need to give additional details if you've lived or worked outside of the UK. You'll also need to give your marital status and details of any partners.

To use this service, you will need to use your own bank or building society account.
